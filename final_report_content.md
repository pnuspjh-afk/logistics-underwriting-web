# [최종 발표 보고서] 물류 자산 투자 분석 통합 플랫폼 구축 프로젝트

## 1. 프로젝트 개요
### 1.1 배경 및 목적
최근 물류 부동산 시장의 변동성 확대에 따라 정밀한 투자 심사(Underwriting)의 중요성이 증대되었습니다. 기존의 엑셀 기반 분석은 정교하지만 공유와 시각화에 한계가 있었고, 웹 기반 대시보드는 사용자 편의성은 높지만 복잡한 시나리오 분석 기능이 부족했습니다. 본 프로젝트는 이 두 가지 장점을 결합하여 **"고성능 분석 엔진과 현대적 UI가 통합된 전문 물류 언더라이팅 플랫폼"**을 구축하는 것을 목적으로 합니다.

### 1.2 프로젝트 성과 요약
- **분석의 전문성 확보**: Python 기반의 금융 공학 엔진을 통해 IRR, DSCR, LTV 등 핵심 지표 산출 로직의 무결성 확보.
- **사용자 경험(UX) 혁신**: React 기반의 다크 모드 대시보드를 통해 실시간 시뮬레이션 환경 제공.
- **업무 자동화**: 분석 결과의 엑셀/PDF 리포트 출력 및 AI 기반 투자 메모 생성 기능 통합.

---

## 2. 주요 기능 및 기술적 특징
### 2.1 통합 아키텍처 (Hybrid Stack)
- **Backend (FastAPI)**: 계산 집약적인 금융 로직과 리포트 생성 기능을 API 서비스화하여 고성능 데이터 처리 구현.
- **Frontend (React/TypeScript)**: 상태 관리 최적화를 통해 슬라이더 조작 시 실시간으로 분석 결과가 갱신되는 인터랙티브 UI 구현.

### 2.2 핵심 분석 기능
- **실시간 UW 시뮬레이션**: 매입가, 임대료, 금리 등 주요 변수 변경 시 즉각적으로 IRR(내부수익률)과 DSCR(부채상환배수) 재계산.
- **멀티 시나리오 분석**: Base, Downside(최악), Upside(최선) 시나리오를 자동으로 생성하고 차트를 통해 직관적으로 비교.
- **2D 민감도 매트릭스**: 공실률과 매각 캡레이트의 복합적인 변화에 따른 수익률 변동폭을 정밀하게 산출.

### 2.3 지능형 리포팅 시스템
- **자동 투자 메모(Investment Memo)**: 분석 결과를 텍스트 마이닝하여 해당 딜의 강점(Merits)과 리스크(Risks)를 국문 리포트 형태로 자동 생성.
- **One-Click Export**: 실무 보고용으로 즉시 사용 가능한 형태의 엑셀(.xlsx) 및 PDF 보고서 생성 지원.

## 3. 기술적 도전 및 해결 과정
- **CORS 및 API 통합**: 서로 다른 기술 스택(Python-React) 간의 원활한 통신을 위해 CORS 정책을 설정하고, Pydantic 모델을 통한 데이터 타입 검증으로 통신 안정성 확보.
- **복잡한 금융 수식 구현**: Newton-Raphson 알고리즘을 활용한 IRR 계산 로직을 Python 백엔드로 일원화하여 계산 오차 제거 및 성능 최적화.
- **리포트 렌더링**: `fpdf2`와 `openpyxl`을 활용하여 서버 사이드에서 전문적인 서식의 문서를 동적으로 생성하는 엔진 구축.

## 4. 기대 효과 및 향후 발전 방향
### 4.1 기대 효과
- **의사결정 속도 향상**: 기존 수작업 위주의 UW 업무 시간을 획기적으로 단축.
- **분석의 객관성**: 표준화된 로직 사용으로 담당자별 분석 편차 제거 및 심사 품질 상향 평준화.

### 4.2 향후 로드맵
- **데이터베이스 연동**: 과거 딜 데이터를 축적하여 권역별 임대료 시세 및 Cap Rate 추이 분석 기능 추가.
- **GIS 기반 시각화**: 지도 API를 연동하여 대상 자산의 입지적 특성(IC 인접도, 교통망) 자동 분석.
- **협업 툴 연동**: 팀별 시나리오 공유 및 승인 프로세스 디지털화.
